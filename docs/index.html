Welcome to Sitepress

Website files

All the website files are in /Users/janb/Documents/Coding/TestPress. If you've built stuff on Rails, you'll feel right at home because Sitepress is actually running on Rails.

Assets

There's an asset pipeline at /Users/janb/Documents/Coding/TestPress/assets where you can put all of your images, stylesheets, and scripts, which will be compiled and fingerprinted for your deployment via Sprockets.

Compile

When you're ready to build the website, run sitepress compile and the website files and assets will all be built and compiled into the /Users/janb/Documents/Coding/TestPress/build path.

Deploy

An example Rakefile is included at /Users/janb/Documents/Coding/TestPress/Rakefile that deploys the build to an AWS S3 bucket. You can customize it to fit your needs or use a completely different deployment pipeline. Whatever floats your boat.

Documentation

Check out the Sitepress website for help on getting started and documentation. Since Sitepress is also built on top of Rails, most of the Rails view helpers work too.

Running version Sitepress 4.0.5. Visit sitepress.cc for documentation, help, and updates.
